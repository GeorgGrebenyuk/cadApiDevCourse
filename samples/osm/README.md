Открытые геоданные OpenStreetMap для вашей ГИС  
==============================================

## Общая информация  

Получить свежие данные OpenStreetMap в удобном формате можно с помощью data.nextgis.com  

Источник:  
   * OpenStreetMap (https://osm.org) – открытая база геоданных и сообщество редакторов, поддерживающих её в актуальном состоянии.  
   * Коды ОКТМО в слоях административных границ и населенных пунктов - https://rosstat.gov.ru/opendata/7708234640-oktmo
